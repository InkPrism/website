Info (Safer Browser)


JavaScript is a very powerful coding language. But with big advantages, there are big dangers, too.
In the past Firefox and other browsers have had many security vulnerabilities that have allowed malicious JavaScript to do bad things to a user's system.
(In many cases native code could be run through JavaScript, meaning a website could potentially do anything to your computer.)
There is also a possibility of cross-site scripting attacks.
